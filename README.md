# uniqlo_searcher

## 主旨

- 在逛Uniqlo時遇到喜歡的衣服、褲子、外套，當下直接買下去之後卻發現隔天開始特價？
- 因此需要一個告訴使用者當下商品價格情況的小偵探！！

## 主要功能

- 本服務會提供：
1. 約半年內的高低價以及現貨價格提供參考
2. 提供追蹤清單服務，小偵探會在特價時第一時間通知您唷！！

---

🔥 小偵探這就來教大家掌握商品行蹤 🔥

下面步驟照著來ᕦ(｀А´)ᕤ

1️⃣  輸入商品編號６碼(參考下圖)，我會告訴您該商品的最新價格💰
